Each unit in this subsection contains a instance of a "built-in advanced component".

Technically speaking, that means each unit contains a usage of an XBlock type which comes installed with edx-platform (either by being coded into edx-platform directly or by installation from edx-platform's base requirements) but is not in one of the standard components categories ("Problem", "Text", "Open Response", "Video", or "Discussion").

The title of each unit is in the format "Example: $TAG" where $TAG is the lowercase XBlock tag used to install and internally reference the block type.

Heads up! Several blocks in this subsection have just been added using their default template and haven't yet been configured in a useful way. Contributors, feel free to propose changes to the configurations of the blocks in this subsection to make the more useful.
