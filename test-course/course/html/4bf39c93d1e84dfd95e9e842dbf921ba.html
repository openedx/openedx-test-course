This subsection contains units whose blocks are restricted to members of one or more content groups. We've created three content groups for this section:

  • Test Content Group X
  • Test Content Group Y
  • Test Content Group Z

Please note that content groups are a content-level concept, not an learner-level one. Specifically, they content groups themselves do not contain learners. In order to test out this subsection, you will need to associate one or more cohorts (an LMS data model grouping course learners together) with the content groups. If someone has not already done so, it is easy to set up cohorts:

  • Navigate to this course in LMS by clicking View Live from Studio.
  • Click the Instructor course tab to navigate to the Instructor Dashboard.
  • Click the Cohorts tab within the Instructor Dashboard.
  • Click Add Cohort.
    • Give it a name (for example "Cohort X") and associate it with a content group (for example "Test Content Group X").
    • Select "Manual Assignment" and add a learner to it by email address.
    • Optionally repeat for Test Content Groups Y and Z.

Lastly, it should be noted that course staff (including global administrators) are implicitly considered to be part of every content group, whether or not they have been added to the associated cohorts.
